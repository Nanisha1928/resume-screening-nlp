# 🚀 Resume Screening System (NLP)

## 📌 Project Overview
This project is an **NLP-based Resume Screening System** that automatically classifies resumes into different job categories using machine learning.

It simulates a real-world recruitment system where resumes are filtered based on their content.

---

## 🎯 Key Features
- 🧹 Text preprocessing using NLTK
- 🔍 Feature extraction using TF-IDF (with n-grams)
- 🤖 Multi-class classification using LinearSVC
- ⚖️ Handles class imbalance
- 📊 Model evaluation with accuracy & classification report
- 🌐 Interactive UI using Streamlit

---

## 🛠 Tech Stack
- Python
- Scikit-learn
- NLTK
- Pandas, NumPy
- Streamlit

---

## 📊 Model Performance
- **Accuracy:** ~71%
- Improved performance using:
  - TF-IDF with n-grams
  - Class balancing
  - SVM (LinearSVC)
